# Java Enterprise Microservices System

This project demonstrates a scalable enterprise backend system built using Java and Spring Boot.

## Technologies Used
- Java
- Spring Boot
- REST APIs
- Docker
- Microservices Architecture
- SQL Database
- CI/CD

## Features
- RESTful API endpoints
- Dependency Injection
- Containerized microservices
- Database integration
- Scalable backend architecture

## Purpose
This project demonstrates enterprise application development practices used in modern distributed systems.
